# 🐹 The Ultimate Go Learning Hub

This repository contains your structured, end-to-end journey to mastering the Go (Golang) programming language.

## 📁 Repository Structure

Each directory represents a specific phase and step. Inside each step folder, you will find:
*   `README.md`: Explaining the concepts, analogies, and the mini-challenges.
*   `main.go`: Starter code template.
*   `go.mod`: Initialized module for instant running.

## 🏃 Quick Command Cheat-sheet

To run the program in any folder:
```bash
go run main.go
```

To run tests in a folder:
```bash
go test -v
```
