# Project 2: High-Performance REST API (Vanilla Go) 🌐

Build a REST API to manage a list of Todo items using Go's built-in standard library (`net/http`).

## Requirements
1. Use vanilla `net/http` router (Go 1.22+ supports pattern matching like `/todos/{id}`).
2. Structs for JSON encoding/decoding.
3. Support GET `/todos` and POST `/todos`.
