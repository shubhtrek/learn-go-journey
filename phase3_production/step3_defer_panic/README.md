# Step 3.3: Defer, Panic, & Recover 🧹

Go's resource management and exception handling keywords.

## Key Concepts
1. **`defer`**: Schedules a function call to run *after* the surrounding function exits. Used for cleanups.
2. **`panic`**: Stops normal execution and crashes the app.
3. **`recover`**: Captures a panic and allows the program to continue running safely.

## How to Run
```bash
go run main.go
```
