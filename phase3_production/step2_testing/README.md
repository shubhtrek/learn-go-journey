# Step 3.2: Unit Testing 🧪

Go comes with a built-in testing tool. Test files must end with `_test.go`.

## Key Concepts
1. **Test Signature**: `func TestXxx(t *testing.T)`
2. **Table-driven tests**: Writing elegant slices of test cases.
3. **Running tests**: `go test -v`

## How to Run Tests
```bash
go test -v
```
