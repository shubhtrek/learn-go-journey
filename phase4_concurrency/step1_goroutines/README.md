# Step 4.1: Goroutines 🏃

Goroutines are lightweight threads managed by the Go runtime.

## Key Concepts
1. **`go` Keyword**: Prefixing a function call with `go` starts it in a new goroutine.
2. **Execution**: The main goroutine doesn't wait for background goroutines to finish automatically.

## How to Run
```bash
go run main.go
```
