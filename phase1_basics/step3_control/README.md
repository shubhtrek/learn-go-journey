# Step 1.3: Control Flow (The One Loop) 🔀

Go has very simple control structures. It only has one loop: `for`!

## Key Concepts
1. **`if/else`**: No parentheses around the condition, but curly braces `{}` are mandatory.
2. **`switch`**: No `break` statement needed; Go break statements are implicit.
3. **`for`**: Go does not have `while` or `do-while` loops. A `for` loop without conditions is an infinite loop.

## How to Run
```bash
go run main.go
```
