# Step 1.4: Functions & Multiple Returns 📞

Functions in Go are clean, support multiple return values, and pass parameters by value.

## Analogy
Ordering a pizza: the delivery person returns the pizza AND the receipt. Go functions can return the result AND an error at the same time.

## Key Concepts
1. **Defining functions**: Use the `func` keyword.
2. **Multiple returns**: Specify return types inside parentheses: `func divide(a, b int) (int, error)`
3. **Named returns**: You can name the return values in the signature.

## How to Run
```bash
go run main.go
```
