# Step 1.1: Hello World & Tooling 🚀

Welcome to the first step of your Go journey! Go programs are organized in packages, and execution starts in package `main`.

## Analogy
`go mod init` is like renting an empty apartment and getting the keys. The Go files we write are the furniture we put inside.

## Key Concepts
1. **`package main`**: Declares that this file is the entry point of the application.
2. **`import "fmt"`**: Imports the Format package for printing text to the console.
3. **`func main()`**: The starting block. Code inside here runs when the program starts.

## How to Run
In this directory, open your terminal and run:
```bash
go run main.go
```
