# Step 2.2: Maps (Key-Value Pairs) 🗺️

Maps are Go's built-in hash tables/dictionaries.

## Key Concepts
1. **Creation**: Use `make(map[keyType]valueType)`.
2. **Check Existence**: The comma-ok idiom `value, ok := map[key]`.
3. **Deletion**: `delete(map, key)`.

## How to Run
```bash
go run main.go
```
