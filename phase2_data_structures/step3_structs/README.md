# Step 2.3: Structs & Methods 🏗️

Go doesn't have classes. It uses `struct` to represent custom schemas and `methods` to add behavior.

## Key Concepts
1. **Struct Definition**: Group fields together.
2. **Value Receivers**: Works on a copy of the struct.
3. **Pointer Receivers**: Can modify fields inside the struct.

## How to Run
```bash
go run main.go
```
