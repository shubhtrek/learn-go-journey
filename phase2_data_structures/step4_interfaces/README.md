# Step 2.4: Interfaces (Go's Secret Weapon) 🔌

Interfaces define a set of methods. In Go, structs don't declare they implement an interface. They do it implicitly!

## Key Concepts
1. **Implicit Implementation**: If it defines all interface methods, Go automatically treats it as that type.
2. **Decoupling**: Helps write testable code by allowing mock types.

## How to Run
```bash
go run main.go
```
